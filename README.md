# Proyecto frontend InaqStock

Desarrollado en Vite + React + TailwindCSS, con tecnología optimizada y avanzada bassada en typescript.

## Tecnologías

- React
- TailwindCSS
- Vite
- Typescript

## Desarrollo

### Instalación

```bash
yarn install
```

### Inicio modo desarrollo

```bash
yarn dev
```

### Compilación

```bash
yarn build
```

### Inicio modo compilación

```bash
yarn start
```
